## Overview

**bothBook** is a documentation and knowledge-sharing website. It organizes pages of notes, references, and short guides in a simple HTML-based structure so it stays fast, readable, and easy to maintain.

## Visit

- Live site: **https://b9b4ymin.github.io/bothBook/**

## Tech

- **HTML** (static site)
- **GitHub Pages** for hosting

## Run Locally

### Option 1 — Open directly

1. Clone or download the repository
2. Open `index.html` in your browser

### Option 2 — Local server (recommended)

**Python**

```bash
python -m http.server 8000
```

Open: `http://localhost:8000`

**Node (serve)**

```bash
npx serve .
```
